Module: requestsController

Este módulo contiene la lista global de solicitudes de cola y las funciones que la manejan para solicitar una cola, cancelar una solicitud y para ser usado por el módulo controllers/algorithmController en la recomendación de solicitudes de cola a un usuario conductor del sistema PideCola.

(inner, constant) stops :Array.<Stop>

Lista global de paradas donde se almacenan ordenadas como una cola First In- First Out, las solicitudes de cola por parada de destino o de origen. El orden de las paradas obdece a la distancia de cada parada desde la USB.
